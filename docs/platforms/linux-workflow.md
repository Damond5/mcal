### Running Tests

Use `fvm flutter test` to run tests on the Linux platform:

```bash
fvm flutter test
```

To run tests with coverage:

```bash
fvm flutter test --coverage
```

To run a specific test file:

```bash
fvm flutter test test/path/to/test_file_test.dart
```
